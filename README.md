# Petit jeu 2D avec Godot

## Description
J’ai créé ce petit jeu en 2D pour m’entraîner à utiliser Godot et comprendre les bases du développement de jeu.

Le principe est très simple : on contrôle un carré et le but est de récupérer des ronds qui apparaissent à l’écran.

## Gameplay
- On déplace un carré
- Des ronds apparaissent
- Il faut en récupérer le plus possible

## Objectif
Faire le meilleur score possible en récupérant un maximum de ronds.

## Pourquoi ce projet ?
C’est un projet d’entraînement pour :
- apprendre à utiliser Godot
- comprendre les collisions
- gérer les déplacements du joueur
- manipuler des scènes et des scripts

# Small 2D Game with Godot

## Description
I created this small 2D game to practice using Godot and learn the basics of game development.

The concept is very simple: you control a square and your goal is to collect circles that appear on the screen.

## Gameplay
- You control a square
- Circles appear on the screen
- You have to collect as many as possible

## Objective
Get the highest score by collecting as many circles as you can.

## Why this project?
This project is mainly for learning:
- how to use Godot
- handling collisions
- managing player movement
- working with scenes and scripts

## Installation
1. Clone the project
2. Open it with Godot
3. Run the main scene

## Possible improvements
- Add enemies or obstacles
- Add sound effects
- Improve visuals
- Add a more advanced scoring system
